# Sports vs Politics Text Classification

## Student Details

Roll Number: B22CS045

---

## Task

Classify news articles into SPORT or POLITICS using machine learning.

---

## Dataset

20 Newsgroups dataset loaded using sklearn API.

Categories used:

* rec.sport.baseball
* rec.sport.hockey
* talk.politics.mideast
* talk.politics.misc

Total documents: 3708

---

## Models Used

* Naive Bayes
* Logistic Regression
* Support Vector Machine (SVM)

Best accuracy achieved by SVM (~95%)

---

Install dependencies:

```
pip install scikit-learn matplotlib numpy
```

```
python3 code/B22CS045_prob4.py
```

Dataset downloads automatically.
